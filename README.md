Perceraian merupakan salah satu permasalahan sosial yang dapat dipengaruhi oleh berbagai aspek, terutama kondisi ekonomi masyarakat. Oleh karena itu, tujuan dari tugas ini adalah untuk menganalisis hubungan antara faktor ekonomi seperti pengeluaran per kapita, tingkat pengangguran terbuka, dan persentase penduduk miskin terhadap jumlah perceraian di Indonesia. Selain itu, tugas ini bertujuan untuk menerapkan metode regresi linear berganda dalam membangun model prediksi jumlah perceraian berdasarkan data BPS tahun 2022–2025, sehingga dapat memberikan gambaran mengenai pengaruh faktor ekonomi terhadap tingkat perceraian serta kemampuan model dalam melakukan prediksi.
